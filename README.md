# CreditWise Loan Approval System

A machine learning project that predicts loan approval status based on applicant financial and personal details.

---

## Overview

The CreditWise Loan Approval System is designed to assist in automating loan approval decisions using data analysis and machine learning techniques. It reduces manual effort and improves decision consistency by identifying patterns in applicant data.

---

## Key Features

* Data preprocessing and handling missing values
* Exploratory data analysis (EDA)
* Feature engineering
* Model training and evaluation
* Loan approval prediction

---

## Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## Getting Started

Clone the repository:

```
git clone https://github.com/your-username/CreditWise-Loan-System.git
```

Navigate to the project folder:

```
cd CreditWise-Loan-System
```

Install required libraries:

```
pip install pandas numpy matplotlib seaborn scikit-learn
```

Run the notebook:

```
jupyter notebook
```
